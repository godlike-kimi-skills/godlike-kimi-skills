## 四、用户运营

### 4.1 用户分层

**核心贡献者 (1%)**
- 定义：提交高质量PR，持续贡献
- 运营：VIP待遇，直接联系，专属徽章

**活跃用户 (10%)**
- 定义：经常使用，提Issue/建议
- 运营：快速响应，纳入决策讨论

**普通用户 (60%)**
- 定义：使用skills，偶尔反馈
- 运营：定期内容触达，活动通知

**旁观者 (29%)**
- 定义：Star了仓库，未深度使用
- 运营：降低门槛，入门内容

### 4.2 激励机制

**贡献者激励**
- 贡献者名单展示 (README首页)
- 专属贡献者徽章
- 月度最佳贡献者评选
- 实物奖励 (周边T恤/贴纸)

**使用者激励**
- 使用反馈奖励 (提Issue送周边)
- 优质案例征集
- 抽奖活动

## 六、危机预案

### 6.1 负面反馈处理

**收到负面评价**
1. 第一时间回复，表示重视
2. 了解具体问题
3. 给出解决方案或时间表
4. 公开透明处理

**模板回复**
```
感谢反馈！🙏

您提到的问题我们非常重视，
能否详细描述一下具体情况？

我们会尽快处理，预计X天内解决。

如有紧急需求，可以联系：xxx
```

### 6.2 竞对攻击应对

**情况：竞对抹黑或抄袭**
- 保持冷静，不直接冲突
- 突出自身差异化优势
- 用数据和案例说话
- 社区口碑自然发酵
