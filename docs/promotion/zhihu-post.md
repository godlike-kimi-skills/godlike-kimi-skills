# 我用AI在GitHub上建了一座"月宫"

![月神技能封面图](封面图链接)

## 一、为什么做这个项目？

**2026年2月**，Kimi CLI发布了。

作为月之暗面 (Moonshot AI) 的编程助手产品，Kimi CLI很强大，但有个致命问题：

> **官方只内置了2个skills。**

而且文档全是英文。

作为中文开发者，我很难用上这些AI能力。

### 问题的严重性

对比一下竞品：
- **Claude Code**: 50+内置skills
- **OpenClaw**: 5000+社区skills
- **Kimi CLI**: 2个skills

差距不是一般的大。

## 二、为什么选择"嫦娥月兔"主题？

你可能会问：做个Skills仓库，为什么要用嫦娥月兔这种"不搭边"的主题？

因为这不是不搭边，这是**命中注定的契合**。

### 品牌契合度分析

```
Kimi "月之暗面" (Moonshot AI)
    ↓
"月" → 月亮 → 嫦娥奔月
"暗面" → 神秘 → 玉兔捣药
```

**月之暗面 × 嫦娥月兔 = 天然的文化共鸣**

而且：
1. **记忆点** - 比"Awesome Kimi Skills"好记100倍
2. **差异化** - 和其他AI项目形成鲜明对比
3. **文化认同** - 中国开发者看到会会心一笑

### 精选Skill

**bayesian-decision** - 贝叶斯决策

适用场景：

## 四、AI自主运营是什么？

这是全球首个**完全由AI运营**的开源项目。

### 人机协作模式

```
Wang Johnny (人类)
    ↓ 提供战略方向
Kimi Code CLI AI Agents
    ↓ 执行任务
代码 / 文档 / 质量检查 / 日常维护
```

### AI具体做什么？

1. **代码生成** - AI编写skills核心代码
2. **文档撰写** - AI生成SKILL.md双语文档
3. **质量检查** - AI自动运行质量评分
4. **竞对扫描** - AI每日扫描竞品新skills
5. **社区回复** - AI辅助回复Issue和PR

人类只需要：
- 制定战略方向
- 审核关键决策
- 处理特殊情况

### 效率提升

传统开源项目维护：
- 需要3-5人全职团队
- 每天8小时工作量
- 成本高，效率低

AI自主运营：
- 1人监督 + AI执行
- 24/7不间断工作
- 成本低，效率高

## 五、质量认证体系

不是简单的"好用/不好用"，而是**5级中国等级制**：

| 等级 | 徽章 | 名称 | 标准 | 占比 |
|------|------|------|------|------|
| AAA | 🏆 | 金印 | 95-100分 | 5% |
| AA | 🥈 | 银牌 | 80-94分 | 20% |
| A | 🥉 | 铜牌 | 60-79分 | 50% |
| B | 📜 | 玉简 | 40-59分 | 20% |
| C | 🏷️ | 木牌 | <40分 | 5% |

### 评分维度

- 功能完整性 (25%)
- 代码质量 (25%)
- 文档质量 (20%)
- 安全性 (15%)
- 规范符合度 (10%)
- 社区反馈 (5%)

每个skill都有质量报告，透明可查。
