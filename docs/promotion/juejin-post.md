**月神技能 (Godlike Kimi Skills)** 是专为中文开发者打造的 Kimi Code CLI Skills 仓库。

## 精选Skill推荐

### 🏆 金印级 (大师推荐)

**bayesian-decision** - 贝叶斯决策
```bash
kimi install bayesian-decision
```
商业决策的核武器。基于贝叶斯认知判定框架，对商业/技术决策做置信度校验、偏差审计。

使用场景：
- 投资决策前的风险评估
- 产品方案的选择分析
- 技术架构的决策支持

**coding-agent** - AI编程助手
```bash
kimi install coding-agent
```
比Copilot更懂中文开发者的AI助手。

### 🥈 银牌级 (优秀品质)

- **git-automation** - Git自动化，告别繁琐命令
- **a-stock-tracker** - A股追踪，程序员也要理财
- **file-organizer** - 智能文件整理，拯救乱糟糟的桌面
- **brave-search** - Brave搜索，隐私保护

### 🥉 铜牌级 (实用工具)

- **crypto-wallet** - 加密货币钱包
- **password-manager** - 密码管理
- **long-term-memory** - 长期记忆，跨会话保持

## 快速开始

### 一键安装

```bash
curl -fsSL https://godlike-kimi.cn/install.sh | bash
```

### 安装单个Skill

```bash
kimi install bayesian-decision
kimi install coding-agent
kimi install a-stock-tracker
```

### 克隆全部

```bash
git clone https://github.com/godlike-kimi-skills/godlike-kimi-skills.git ~/.kimi/skills/
```
