## 快速开始

### 一键安装

```bash
curl -fsSL https://godlike-kimi.cn/install.sh | bash
```

### 安装单个Skill

```bash
kimi install bayesian-decision
kimi install coding-agent
kimi install a-stock-tracker
```

### 克隆全部

```bash
git clone https://github.com/godlike-kimi-skills/godlike-kimi-skills.git ~/.kimi/skills/
```
