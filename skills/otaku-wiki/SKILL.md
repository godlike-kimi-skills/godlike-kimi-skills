# Otaku Wiki

番剧/角色百科问答。

## Features

- 番剧信息查询
- 角色资料搜索
- 声优信息
- 作品关联

## Usage

```bash
# 查询番剧
python D:/kimi/skills/otaku-wiki/scripts/wiki.py anime "鬼灭之刃"

# 查询角色
python D:/kimi/skills/otaku-wiki/scripts/wiki.py character "炭治郎"

# 查询声优
python D:/kimi/skills/otaku-wiki/scripts/wiki.py seiyuu "花泽香菜"
```
