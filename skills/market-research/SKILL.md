# Market Research

**市场研究与情报收集专家** - 借鉴 Gartner, CB Insights, Crunchbase

自动搜索行业数据、竞争对手信息、市场规模、增长趋势。输出结构化市场报告。

---

## 核心特性

### 📊 研究维度

| 维度 | 说明 |
|------|------|
| **市场规模** | TAM/SAM/SOM 分析 |
| **竞争格局** | 主要玩家、市场份额 |
| **增长趋势** | CAGR、趋势预测 |
| **融资情报** | 投融资动态、独角兽追踪 |

### 📈 报告模板

```
市场报告:
├── 执行摘要
├── 市场概况
├── 竞争分析
├── 趋势预测
├── 机会与风险
└── 数据来源
```

---

## 使用方法

### 行业研究
```bash
# 基础市场分析
market-research analyze --industry "AI芯片" --depth comprehensive

# 竞争对手分析
market-research competitors --company "OpenAI"

# 融资情报
market-research funding --sector "fintech" --stage series-a

# 生成报告
market-research report --topic "SaaS trends 2025" --output report.md
```

## 战略分析框架 (Strategic Framework)

### 🎯 战略定位

```
愿景: 成为AI时代领先的"智能市场情报与战略咨询平台"
使命: 利用AI技术 democratize 战略咨询
差异化: AI原生 + 实时更新 + 可执行洞察 + 垂直深耕
```

### 🔍 波特五力分析

| 力量 | 评估 | 战略含义 |
|------|------|----------|
| **新进入者威胁** | 中偏高 | 建立数据护城河，深化行业know-how |
| **供应商议价能力** | 中 | 数据源多元化，建立自有采集能力 |
| **买方议价能力** | 高 | 深度差异化，提高转换成本 |
| **替代品威胁** | 高 | 人机协同，提供Actionable Insights |
| **行业内竞争** | 激烈 | 聚焦垂直领域，避免正面竞争 |

### 📊 SWOT 分析

```
优势 (Strengths):
├── 成熟的Skill框架和基础功能
├── 参考Gartner/CB Insights的方法论
├── Kimi CLI生态系统深度集成
└── 结构化的报告模板

劣势 (Weaknesses):
├── AI分析能力有限
├── 缺乏深度洞察和可执行建议
├── 垂直行业Know-how不足
└── 缺乏客户成功案例

机会 (Opportunities):
├── AI技术带来的效率提升
├── 企业数字化转型需求增长
├── 与proactive-agent等Skills协同
└── 中国市场的本土数据优势

威胁 (Threats):
├── 大语言模型直接提供分析能力
├── 数据供应商提价或限制访问
├── AI原生竞争者快速崛起
└── 经济下行导致预算削减
```
