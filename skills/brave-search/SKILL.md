### 决策算法

```python
def select_search_engine(query, user_preferences):
    """基于期望效用的搜索引擎选择"""
    
    # 分析查询类型
    query_type = classify_query(query)
    
    # 获取该类型的权重
    quality_weight, privacy_weight = get_weights(query_type)
    
    # 计算各引擎的EU
    brave_eu = calculate_eu(
        quality=0.85, 
        privacy=0.95, 
        weights=(quality_weight, privacy_weight)
    )
    google_eu = calculate_eu(
        quality=0.90, 
        privacy=0.30, 
        weights=(quality_weight, privacy_weight)
    )
    
    # 考虑用户偏好
    if user_preferences.privacy_focused:
        brave_eu *= 1.2
    
    # 选择EU更高的
    return 'brave' if brave_eu > google_eu else 'google'
```

## 因果推断分析

### 隐私保护的因果效应

**真实因果图:**

```
                    用户隐私意识 (混淆变量)
                         ↓
        ┌────────────────┼────────────────┐
        ↓                ↓                ↓
选择Brave ←──── 技术能力 ────→ 其他隐私措施
        ↓                ↓                ↓
        └────────→ 实际隐私水平 ←─────────┘
                      ↓
              数据泄露风险
                      ↑
    ┌─────────────────┼─────────────────┐
    ↓                 ↓                 ↓
搜索引擎追踪    网络监控        应用权限滥用
```

**关键洞察:** 用户隐私意识是混淆变量，高隐私意识用户选择Brave的同时也采取其他隐私措施。

**干预分析:**

```
观察: P(隐私泄露|使用Brave) = 低
干预: P(隐私泄露|do(使用Brave))

估计因果效应:
├─ 观察值偏低：Brave用户本身更重视隐私
├─ 干预值：普通用户切换到Brave的效果
└─ 估计：do(使用Brave) 降低追踪概率 60-80%
```

**边界条件:**
- 已登录Google账号时，Brave的保护效果受限
- 使用其他追踪服务（社交媒体插件）会抵消效果
- 企业网络监控不受搜索引擎选择影响

```python
def blind_quality_test():
    queries = load_neutral_queries(100)
    results = {}
    
    for query in queries:
        brave_results = search(query, engine='brave', branding='removed')
        google_results = search(query, engine='google', branding='removed')
        
        # 让用户评分（不知道来源）
        results[query] = user_evaluate(brave_results, google_results)
    
    # 证伪标准: Brave评分 < Google的85%
    brave_avg = mean([r['brave'] for r in results.values()])
    google_avg = mean([r['google'] for r in results.values()])
    
    if brave_avg < 0.85 * google_avg:
        return "声明被证伪"
```
