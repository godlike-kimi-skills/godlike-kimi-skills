# Critical Thinking - 自我分析

## 概况
| 属性 | 内容 |
|------|------|
| 质量 | ⭐⭐⭐⭐ (4/5) |
| 功能 | 逻辑谬误检测与批判思维 |
| 特色 | 7大谬误+苏格拉底提问+逆向思维 |

## 元认知分析 (用批判思维分析批判思维)

### 1. 逻辑自洽性 - 90%
✅ **自洽**: 用清单形式避免自身谬误
✅ **实用**: 每项谬误都有"Fix"
✅ **完整**: 覆盖主要认知偏差

### 2. Falsifiability (可证伪性) - 60%
**问题**: "避免这些谬误就能做出好决策吗?"
- 不可证伪: 没有定义"好决策"标准
- 幸存者偏差: 清单本身是否完整?

**改进**:
```markdown
## 批判思维的有效性测试

测试1: 使用该清单的决策 vs 未使用的
- 指标: 决策后悔率、结果满意度
- 样本: N>100次决策

测试2: 7项覆盖度
- 检查真实谬误案例中未被覆盖的比例
- 目标: <10%遗漏
```

### 3. Completeness (完整性) - 50%
**只覆盖7个谬误，实际更多**:
- 基本归因错误
- 锚定效应
- 可得性启发
- 框架效应
- ...

**建议扩展至20+常见谬误**

## 关键洞察
这是一个**元技能** (关于思考的思考)，应当:
1. 与其他所有skills交叉引用
2. 作为质量检查标准
3. 嵌入决策流程

## 改进优先级: P1 (重要增强)
- 扩展谬误清单至20+
- 添加可证伪性测试
- 创建"自检清单"用于自检
