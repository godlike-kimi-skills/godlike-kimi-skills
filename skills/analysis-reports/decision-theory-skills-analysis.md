## AI Agent项目投资决策

Step 1 - Bayesian更新:
```
先验: AI Agent项目成功率 = 20%
证据:
  - 团队有经验 (+15%)
  - 市场时机好 (+10%)
  - 竞争激烈 (-5%)
后验: 约40%成功率
```

Step 2 - Expected Utility:
```
成功: $100K收益 × 40% = $40K
失败: -$20K损失 × 60% = -$12K
期望效用: $28K
风险调整(λ=2.25): $28K - 2.25×$12K = $1K
```

```

```
