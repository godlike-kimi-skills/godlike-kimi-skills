## 博弈/进化机制分析

### 1. 核心机制矩阵

| 机制 | 类型 | 数学基础 | 应用场景 |
|------|------|----------|----------|
| **ESS** | 稳定性分析 | 不等式约束系统 | 识别不可入侵策略 |
| **鹰鸽博弈** | 对称博弈 | 混合策略均衡 p*=V/C | 竞争强度决策 |
| **囚徒困境** | 非合作博弈 | 收益矩阵分析 | 合作机制设计 |
| **复制者动态** | 群体演化 | 微分方程 dx/dt | 策略比例预测 |
| **协调博弈** | 均衡选择 | 多均衡分析 | 标准/技术选择 |

### 2. 机制详解

#### ESS (演化稳定策略)
```
双重条件确保稳定性:
1. E(S,S) ≥ E(T,S)    → S对自身是最优响应
2. E(S,T) > E(T,T)    → S能抵御小入侵
```
**本质**: 进化版的纳什均衡，强调动态稳定性

#### 鹰鸽博弈的商业映射
| 博弈参数 | 商业对应 |
|----------|----------|
| V (资源价值) | 市场份额/利润池 |
| C (战斗成本) | 价格战损失/法律成本 |
| p* = V/C | 最优激进/保守比例 |

#### 复制者动态
```
dxᵢ/dt = xᵢ [E(i) - E(avg)]

关键洞察:
- 表现优于平均 → 份额增长
- 表现劣于平均 → 份额衰减
- 系统趋向ESS收敛
```

### 3. 演化轨迹特征

| 特征 | 说明 | 决策含义 |
|------|------|----------|
| **收敛性** | 系统趋向ESS | 可预测长期格局 |
| **历史依赖** | 初始条件影响均衡选择 | 先发优势重要 |
| **突变可能** | 随机扰动跳出局部最优 | 保持创新警惕 |
| **正反馈** | 临界点后的加速变化 | 把握颠覆时机 |
