## 作为Lens的价值
### Multi-Model Analysis
```markdown
## 多模型分析模板

问题: [待分析的Skill/决策]

应用模型:
├─ First Principles: 本质是什么?
├─ Inversion: 如何避免失败?
├─ Second-Order: 长期后果?
├─ Bayesian: 概率更新?
├─ TOC: 约束在哪里?
└─ Network Effects: 如何放大?

综合判断: [多模型交叉验证后的结论]
```

### 98个模型的Lens潜力
每个模型都可以作为一个独立的分析Lens:
- m01 First Principles → 本质分析
- m02 Inversion → 风险识别
- m30 Bottlenecks → 约束识别
- m36 Network Effects → 增长分析
- ...etc
