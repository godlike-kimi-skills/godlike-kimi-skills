## 网络演化分析

## Causal Inference (因果推断) - 30%
**问题**: 知识图谱显示相关性，非因果性

**建议添加**:
```markdown
## 从相关性到因果性

知识图谱局限:
- 显示"A和B常一起出现"
- 不说明"A导致B"还是"B导致A"

改进:
1. 添加时间维度 (先出现→后出现的方向)
2. 干预分析 (改变A，B如何变化)
3. 对照实验设计
```

## 第一性原理 - 60%
**本质**: 知识图谱 = 将隐性知识显性化的工具

**价值公式**:
```
知识图谱价值 = (连接数量 × 连接质量) / 认知负荷
```

**优化方向**:
- 增加连接: 自动提取
- 提升质量: 人工审核
- 降低负荷: 智能过滤

## 优先级: P1
- 添加网络演化分析
- 区分相关性与因果性
- 与其他skills (如causal-inference) 集成
