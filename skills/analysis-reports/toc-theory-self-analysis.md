## 系统动力学分析

### 反馈回路
TOC本身形成一个增强回路:
```
识别约束 → 挖尽约束 → 打破约束 → 新约束出现 → 回到识别
    ↑                                                    ↓
    └─────────────── 系统持续改进 ───────────────────────┘
```

这是一个**持续改进的增强回路(R)**

### 存量-流量图
```
         [约束识别]
              ↓
    [挖尽潜能] → [约束库存] → [打破约束]
              ↑                ↓
              └──[迁就调整]────┘
```

## 第一性原理视角
**TOC的本质**: 系统的产出由最弱环节决定

**第一性要素**:
1. 系统 interconnected
2. 资源有限
3. 局部优化 ≠ 全局优化

**推导**:
→ 必须识别全局约束
→ 局部效率不重要
→ 约束是动态变化的

## 可证伪性评估: 70%

**可证伪陈述**:
- "任何系统至少有一个约束" → 可证伪 (找到无约束系统即证伪)
- "挖尽约束可提升产出" → 可测试

**不可证伪部分**:
- "警惕惯性成为新约束" (模糊定义)
