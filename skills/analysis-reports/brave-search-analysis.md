### 技术特性

- **隐私保护**: 无用户追踪、独立搜索索引、加密连接
- **CLI界面**: 提供命令行工具 `brave-search`
- **输出格式**: 支持JSON输出

### 使用方法示例

```bash
# 网页搜索
brave-search query "quantum computing" --limit 10

# 图片搜索
brave-search images "cat" --output results.json

# 新闻搜索
brave-search news "AI breakthrough" --days 7
```
