## 第一性原理视角
**精益的本质**: 最大化价值创造与资源消耗的比值

**公式**:
```
精益度 = 客户价值 / (时间 + 资源 + 认知负荷)
```

**本Skill的优化**:
- 增加客户价值: 添加更多实战案例
- 减少认知负荷: 简化理论，强化工具
