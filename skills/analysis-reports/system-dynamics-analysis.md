## 🏛️ 系统理论框架

### 核心理论：系统动力学 (System Dynamics)

```
┌─────────────────────────────────────────────────────────────┐
│                系统动力学理论架构                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  建模方法                                                    │
│  ├─ 因果回路图 (Causal Loop Diagrams)                       │
│  │   ├─ 正反馈环 (Reinforcing Loop) → 指数增长             │
│  │   └─ 负反馈环 (Balancing Loop) → 趋于平衡               │
│  └─ 存量流量图 (Stock and Flow)                            │
│      ├─ 存量 (Stock): 累积量                                │
│      └─ 流量 (Flow): 变化率                                 │
│                                                             │
│  系统元素                                                    │
│  ├─ 存量 □ (库存、资金、人口)                               │
│  ├─ 流量 → (流入/流出率)                                    │
│  ├─ 辅助变量 ○ (中间计算)                                   │
│  ├─ 常量 △ (固定参数)                                       │
│  └─ 延迟 ⏱️ (时间滞后)                                      │
│                                                             │
│  典型行为模式                                                │
│  ├─ 指数增长 (正反馈主导)                                   │
│  ├─ 寻的/稳定 (负反馈主导)                                  │
│  ├─ 震荡 (延迟+负反馈)                                      │
│  ├─ 超调崩溃 (增长触及限制)                                 │
│  └─ S型增长 (先指数后饱和)                                  │
│                                                             │
│  仿真步骤                                                    │
│  问题界定 → 因果分析 → 存量流量 → 参数估计 → 模型验证 → 政策实验 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 理论特色

1. **MIT学派**: 基于Jay Forrester的工业动力学传统
2. **仿真导向**: 强调定量建模和模拟
3. **政策评估**: 用于测试不同干预措施效果
4. **结构决定行为**: 系统结构产生特定行为模式

## 🎯 应用场景

### 场景1: 金融风险仿真
```
模型要素:
- 存量: 资产价值、负债水平、流动性储备
- 流入: 收入、投资收益、新融资
- 流出: 支出、债务偿还、损失
- 反馈: 信心 → 投资 → 资产价格 → 信心
```

### 场景2: 市场情绪模拟
```
因果回路:
利好消息 → 乐观情绪 → 买入增加 → 价格上涨 → FOMO → 更多买入
                                            ↓
泡沫破裂 ← 恐慌抛售 ← 价格下跌 ← 获利了结
```

import numpy as np
import matplotlib.pyplot as plt

# 参数
initial_customers = 100
adoption_rate = 0.03
market_size = 10000

# 仿真
customers = [initial_customers]
for t in range(100):
    new_adopters = adoption_rate * customers[-1] * (1 - customers[-1]/market_size)
    customers.append(customers[-1] + new_adopters)

plt.plot(customers)
