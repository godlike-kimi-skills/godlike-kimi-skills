### 2. First Principles (本质思考) - 70%
**本质**: 商业战略 = 在约束条件下创造和捕获价值

**第一性要素**:
1. 价值创造 (效率/创新)
2. 价值捕获 (定价/壁垒)
3. 竞争动态 (博弈)
4. 环境适应 (演化)

**重构建议**:
```markdown
# 第一性战略框架

核心问题: 为什么客户选择我们而非竞品?

答案必须在以下维度至少一个领先:
1. 效率 (成本领先)
2. 创新 (差异化)
3. 锁定 (转换成本)
4. 网络 (生态效应)

否则: 战略不可持续
```

**缺失**: 战略决策的概率框架

