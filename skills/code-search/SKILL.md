# Code Search

Search for code examples and development resources.

## Features

- Search code examples online
- Find library documentation
- Get implementation patterns
- Research best practices

## Usage

### Search Code Examples

```bash
python D:/kimi/skills/code-search/scripts/search.py "python asyncio example"
```

### Search Specific Sites

```bash
# Search Stack Overflow
python D:/kimi/skills/code-search/scripts/search.py "react hooks" --site stackoverflow

# Search GitHub
python D:/kimi/skills/code-search/scripts/search.py "fastapi CRUD" --site github
```

### Save Results

```bash
python D:/kimi/skills/code-search/scripts/search.py "docker compose example" --save results.md
```

## Search Sources

- Stack Overflow
- GitHub repositories
- Official documentation
- Technical blogs
- Developer forums

## Tips

- Be specific in your query
- Use language/framework names
- Include version numbers for accuracy
