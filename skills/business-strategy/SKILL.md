# Business Strategy

**商业战略分析框架** - 借鉴 McKinsey, BCG, Bain 方法论

波特五力、SWOT、PESTEL、商业模式画布等专业分析工具。

---

## 核心框架

### 1. 波特五力分析 (Porter's Five Forces)

| 力量 | 分析要点 | 评分 |
|------|----------|------|
| **供应商议价能力** | 集中度、转换成本、前向整合 | 1-5 |
| **买方议价能力** | 订单规模、价格敏感、后向整合 | 1-5 |
| **新进入者威胁** | 壁垒、资本需求、规模经济 | 1-5 |
| **替代品威胁** | 替代性能、价格性能比 | 1-5 |
| **行业内竞争** | 竞争者数量、增长率、退出壁垒 | 1-5 |

### 2. SWOT 分析

```
            有利               不利
    ┌──────────────────┬──────────────────┐
内部 │ Strengths (优势)  │ Weaknesses (劣势) │
     │ • 核心竞争力      │ • 资源缺口        │
     │ • 品牌优势        │ • 技术落后        │
├────┼──────────────────┼──────────────────┤
外部 │ Opportunities     │ Threats (威胁)    │
     │ (机会)            │ • 新竞争者        │
     │ • 市场增长        │ • 政策变化        │
└────┴──────────────────┴──────────────────┘
```

### 3. PESTEL 分析

| 维度 | 关键因素 | 示例 |
|------|----------|------|
| **P**olitical | 政策、监管、政治稳定性 | 行业法规 |
| **E**conomic | GDP、利率、通胀、汇率 | 经济周期 |
| **S**ocial | 人口、文化、生活方式 | 消费习惯 |
| **T**echnological | 技术变革、创新、自动化 | AI趋势 |
| **E**nvironmental | 环保、气候、可持续性 | 碳中和 |
| **L**egal | 劳动法、知识产权、消费者权益 | 数据保护 |
