# PPTX Creator

Create and edit PowerPoint presentations.

## Features

- Create slides programmatically
- Add text, images, charts
- Apply templates
- Export to PDF

## Usage

```bash
# Create presentation
python D:/kimi/skills/pptx-creator/scripts/pptx.py create "My Presentation"

# Add slide
python D:/kimi/skills/pptx-creator/scripts/pptx.py add-slide "Title" "Content"

# Generate from markdown
python D:/kimi/skills/pptx-creator/scripts/pptx.py from-markdown outline.md
```
