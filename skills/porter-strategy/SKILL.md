# Porter's Competitive Strategy

**创始人**: Michael Porter (1980)  
**核心著作**: 《竞争战略》  
**类型**: 战略与竞争  
**优先级**: P0

---

## 五力模型 (Five Forces)

### 模型框架

```
                 新进入者威胁
                      ↓
    供应商议价能力 → 行业竞争 ← 买方议价能力
                      ↑
                 替代品威胁
```

**五种力量决定行业盈利能力**:

### 1. 新进入者威胁

```
进入壁垒高低决定威胁程度:

高壁垒 (威胁低):
├─ 规模经济
├─ 资本需求
├─ 转换成本
├─ 渠道控制
├─ 政策许可
└─ 品牌忠诚度

低壁垒 (威胁高):
├─ 小规模即可启动
├─ 少量资金即可
├─ 标准化产品
├─ 开放渠道
└─ 无监管限制

战略启示:
├─ 在壁垒高领域建立优势
├─ 或主动抬高壁垒
└─ 警惕壁垒降低的信号
```

### 2. 供应商议价能力

```
供应商强势信号:
├─ 供应商集中度高
├─ 无替代品
├─ 对买方不重要
├─ 转换成本高
├─ 前向一体化威胁
└─ 买方议价能力弱

应对策略:
├─ 标准化输入 (降低依赖)
├─ 多源采购 (分散风险)
├─ 后向一体化 (自己生产)
├─ 长期合约 (锁定供应)
└─ 规模采购 (增强议价)
```

### 3. 买方议价能力

```
买方强势信号:
├─ 买方集中度高
├─ 产品标准化
├─ 转换成本低
├─ 买方可后向一体化
├─ 价格敏感度高
└─ 购买量占比大

应对策略:
├─ 差异化 (降低可比性)
├─ 提高转换成本
├─ 服务增值
├─ 细分市场
└─ 锁定核心客户
```

### 4. 替代品威胁

```
替代威胁评估:

高威胁:
├─ 替代品性能更好
├─ 替代品价格更低
├─ 转换成本低
└─ 买方替代意愿强

低威胁:
├─ 功能差异大
├─ 情感/习惯绑定
├─ 网络效应锁定
└─ 互补生态依赖

战略应对:
├─ 持续创新 (保持领先)
├─ 生态锁定 (增加粘性)
├─ 进入替代领域 (自我替代)
└─ 强调差异化价值
```

### 5. 行业内竞争

```
竞争激烈程度:

高竞争 (利润低):
├─ 竞争者众多/势均力敌
├─ 行业增长缓慢
├─ 固定成本高
├─ 产品同质化
├─ 退出壁垒高
└─ 战略利益重大

低竞争 (利润高):
├─ 集中度高的寡头
├─ 行业快速增长
├─ 成本结构灵活
├─ 高度差异化
├─ 容易退出
└─ 战术性竞争

竞争战略:
├─ 避免价格战
├─ 寻求差异化
├─ 专注于细分市场
└─ 建立转换壁垒
```

## 三种通用战略

### 1. 总成本领先 (Cost Leadership)

```
目标: 成为行业最低成本生产者

实现路径:
├─ 规模经济
├─ 学习曲线
├─ 成本控制
├─ 效率提升
├─ 供应链管理
└─ 垂直整合

风险:
├─ 价格战
├─ 技术变革使规模优势失效
├─ 新进入者学习成本低
└─ 差异化需求增长

适用条件:
├─ 价格敏感市场
├─ 标准化产品
├─ 转换成本低
└─ 买方以价格为导向
```

### 2. 差异化 (Differentiation)

```
目标: 提供独特价值，获取溢价

差异化维度:
├─ 产品特性
├─ 服务质量
├─ 品牌形象
├─ 技术领先
├─ 客户体验
└─ 渠道便利

风险:
├─ 差异化成本过高
├─ 买方不需要差异化
├─ 模仿者复制
└─ 差异化基础被技术颠覆

适用条件:
├─ 买方需求多样化
├─ 差异化可被感知
├─ 愿意支付溢价
└─ 难以模仿
```

### 3. 聚焦 (Focus)

```
目标: 服务特定细分市场

聚焦类型:
├─ 成本聚焦: 在细分市场成本领先
└─ 差异化聚焦: 在细分市场差异化

细分标准:
├─ 地理区域
├─ 客户群体
├─ 产品细分
├─ 应用细分
└─ 价值链环节

风险:
├─ 细分市场消失
├─ 大玩家进入细分市场
├─ 需求趋同化
└─ 过度依赖单一市场
```

## 价值链分析

### 价值链框架

```
         企业基础设施 (财务、计划、法务)
              ↓
         人力资源管理
              ↓
         技术开发
              ↓
         采购
              ↓
┌─────────────────────────────────────────────┐
│  进料物流 → 生产运营 → 出货物流 → 营销销售 → 服务 │
│  (原料)    (制造)    (分销)    (推广)    (售后) │
└─────────────────────────────────────────────┘
              ↓
           利润

主要活动: 直接创造价值
支持活动: 为主要活动提供支撑
```

### 价值链优化

```
分析步骤:
1. 识别每项活动的成本
2. 评估每项活动的价值贡献
3. 与竞争对手比较
4. 识别优势/劣势环节
5. 制定优化方案

优化策略:
├─ 成本降低: 消除不增值活动
├─ 差异化: 强化关键价值活动
├─ 聚焦: 在核心环节建立优势
└─ 外包: 非核心活动外包
```
