# Security Audit

Security auditing and vulnerability scanning.

## Features

- Code security scan
- Dependency check
- Vulnerability detection
- Compliance checking

## Usage

```bash
# Scan codebase
python D:/kimi/skills/sec-audit/scripts/audit.py scan /path/to/code

# Check dependencies
python D:/kimi/skills/sec-audit/scripts/audit.py deps requirements.txt

# Full audit
python D:/kimi/skills/sec-audit/scripts/audit.py full
```
