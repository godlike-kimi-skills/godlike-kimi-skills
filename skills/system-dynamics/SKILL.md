# System Dynamics 系统动力学

**复杂系统行为仿真框架** - 基于 MIT Jay Forrester 方法论

用于金融风险分析、市场情绪模拟、政策传导评估、宏观趋势预测。

---

## 核心要素

### 因果回路图 (Causal Loop Diagrams)

```
正反馈环 (Reinforcing Loop)
    ┌─────────────────────┐
    ↓                     │
  投资增长 → 收益提升 → 更多投资
    └─────────────────────┘
    
负反馈环 (Balancing Loop)
    ┌─────────────────────┐
    ↓                     │
  市场份额 → 竞争加剧 → 份额下降
    └─────────────────────┘
```

### 存量流量图 (Stock and Flow)

```
         流入率 (Flow In)
              ↓
    ┌───────────────────┐
    │    存量 (Stock)    │ ←── 累积量
    │   (如：现金储备)    │
    └───────────────────┘
              ↓
         流出率 (Flow Out)
```

---

## 关键概念

| 概念 | 符号 | 含义 |
|------|------|------|
| **存量** | □ | 累积量（如库存、资金、人口） |
| **流量** | → | 变化率（流入/流出） |
| **辅助变量** | ○ | 中间计算变量 |
| **常量** | △ | 固定参数 |
| **延迟** | ⏱️ | 时间滞后效应 |

## 典型模式

### 系统行为模式库

```
指数增长 (Exponential Growth)
└── 正反馈主导 → 爆发式增长

寻的/稳定 (Goal Seeking)
└── 负反馈主导 → 趋向平衡

震荡 (Oscillation)
└── 延迟 + 负反馈 → 周期波动

超调崩溃 (Overshoot & Collapse)
└── 增长触及限制 → 快速崩溃

S型增长 (S-curve Growth)
└── 先指数增长 → 后趋于饱和
```

## 应用场景

### 金融风险仿真

```
模型要素：
- 存量：资产价值、负债水平、流动性储备
- 流入：收入、投资收益、新融资
- 流出：支出、债务偿还、损失
- 反馈：信心 → 投资 → 资产价格 → 信心

模拟输出：
- 风险传导路径
- 流动性危机触发点
- 政策干预效果
```

### 市场情绪模拟

```
因果回路：
  利好消息 → 乐观情绪 → 买入增加 → 价格上涨 → FOMO → 更多买入
                                      ↓
  泡沫破裂 ← 恐慌抛售 ← 价格下跌 ← 获利了结
  
模拟情景：
- 牛市持续条件
- 拐点预警信号
- 恐慌扩散速度
```

## 仿真步骤

1. **问题界定**: 明确系统边界与研究问题
2. **因果分析**: 绘制因果回路图识别关键反馈
3. **存量流量**: 转化为可量化的存量流量图
4. **参数估计**: 基于历史数据/专家判断设定参数
5. **模型验证**: 回测检验模型有效性
6. **政策实验**: 测试不同干预措施效果
