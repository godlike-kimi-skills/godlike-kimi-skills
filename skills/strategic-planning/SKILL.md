# Strategic Planning 战略规划与优化

**可执行战略制定框架** - 基于 Hoshin Kanri 方针管理与 OKR 方法论

输出可落地的战略地图，而非空泛分析。

---

## 战略制定七步法

```
┌─────────────────────────────────────────────────────────────┐
│ 1. 愿景定义 (Vision)                                         │
│    └── 3-5 年长期目标、使命宣言                              │
├─────────────────────────────────────────────────────────────┤
│ 2. 现状诊断 (Current State)                                  │
│    └── SWOT、差距分析、关键问题识别                          │
├─────────────────────────────────────────────────────────────┤
│ 3. 差距分析 (Gap Analysis)                                   │
│    └── 目标 vs 现状 = 战略差距                               │
├─────────────────────────────────────────────────────────────┤
│ 4. 路径设计 (Strategic Path)                                 │
│    └── 达成目标的战略举措与里程碑                            │
├─────────────────────────────────────────────────────────────┤
│ 5. 资源配置 (Resource Allocation)                            │
│    └── 人、财、物、时间的优先级分配                          │
├─────────────────────────────────────────────────────────────┤
│ 6. 里程碑设定 (Milestones)                                   │
│    └── 关键结果 (KR)、时间表、责任人                         │
├─────────────────────────────────────────────────────────────┤
│ 7. 风险管理 (Risk Management)                                │
│    └── 风险识别、应对预案、监控机制                          │
└─────────────────────────────────────────────────────────────┘
```

## Hoshin Kanri 方针管理

### 方针展开流程

```
                长期愿景 (3-5 年)
                      ↓
              年度方针 (Breakthrough Objectives)
                      ↓
        ┌─────────────┼─────────────┐
        ↓             ↓             ↓
   部门方针 A     部门方针 B     部门方针 C
   (营销)         (产品)         (运营)
        ↓             ↓             ↓
   个人目标      个人目标       个人目标
   (月度审查)    (月度审查)     (月度审查)
```

###  catchball 过程

```
自上而下: 目标分解、方向传递
    ↕
自下而上: 可行性反馈、资源需求
    ↕
横向协调: 跨部门对齐、依赖同步
```

## OKR 整合框架

| 层级 | 时间周期 | 目标数量 | 关键结果/目标 |
|------|----------|----------|---------------|
| **公司级** | 年度 | 3-5 个 | 3-5 个 KR |
| **团队级** | 季度 | 2-3 个 | 2-4 个 KR |
| **个人级** | 季度 | 2-3 个 | 2-4 个 KR |

### OKR 制定原则

- **O (Objective)**: 定性、鼓舞人心、有挑战性
- **KR (Key Results)**: 定量、可衡量、具体
- **信心指数**: 初始 50%（有一定挑战但可实现）
