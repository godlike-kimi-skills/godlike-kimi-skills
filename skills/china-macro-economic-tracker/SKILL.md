# China Macro Economic Tracker

**中国宏观经济追踪器** - 经济指标监测与分析

GDP、CPI、PPI、PMI、利率等核心指标追踪。

---

## 核心特性

### 📊 经济指标

| 指标 | 频率 | 来源 |
|------|------|------|
| **GDP** | 季度 | 国家统计局 |
| **CPI** | 月度 | 国家统计局 |
| **PPI** | 月度 | 国家统计局 |
| **PMI** | 月度 | 统计局/财新 |
| **M2** | 月度 | 央行 |
| **LPR** | 月度 | 央行 |

### 📈 分析功能

```
数据获取 → 趋势分析 → 同比环比 → 预测模型
```

---

## 使用方法

### 查看GDP数据
```bash
china-macro-economic-tracker gdp --period quarterly
```

### 查看通胀数据
```bash
china-macro-economic-tracker cpi --start 2020
```

### 查看货币供应
```bash
china-macro-economic-tracker m2 --chart
```

## 战略分析框架 (Strategic Framework)

### 🎯 战略定位

```
愿景: 成为中国投资者首选的"AI宏观经济策略顾问"
使命: 让每个人都能获得专业级的宏观经济分析
差异化: AI驱动 + 主动预警 + 投资策略导向
```

### 🔍 波特五力分析

| 力量 | 评估 | 战略含义 |
|------|------|----------|
| **新进入者威胁** | 中 | 建立数据验证机制，开发独特预测模型 |
| **供应商议价能力** | 中低 | 多数据源备份，建立本地缓存 |
| **买方议价能力** | 中高 | 差异化定位，提供策略建议而非单纯数据 |
| **替代品威胁** | 高 | 深度分析+主动预警+个性化服务 |
| **行业内竞争** | 中 | 避开Wind等巨头，专注AI差异化 |

### 📊 SWOT 分析

```
优势 (Strengths):
├── 覆盖中国核心宏观经济指标
├── 官方数据源保证准确性
├── Kimi CLI生态集成
└── 结构化分析框架

劣势 (Weaknesses):
├── 缺乏预测模型和深度分析
├── 没有主动预警机制
├── 用户体验待优化
└── 缺乏垂直定制功能

机会 (Opportunities):
├── AI技术提升分析能力
├── proactive-agent主动监控
├── 投资者对宏观策略需求增长
└── 与投资组合管理工具协同

威胁 (Threats):
├── Wind等商业终端竞争
├── 通用AI助手提供基础分析
├── 数据政策变化
└── 经济不确定性
```
