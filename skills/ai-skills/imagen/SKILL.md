# Image Generation Skill

AI image generation assistant.

## Features

- Prompt engineering optimization
- Style and composition guidance
- Image analysis and feedback
- Format and resolution recommendations

## Usage

```bash
# Optimize prompt
python D:/kimi/skills/ai-skills/imagen/scripts/imagen.py prompt "a beautiful sunset"

# Analyze image concept
python D:/kimi/skills/ai-skills/imagen/scripts/imagen.py analyze "cyberpunk city"

# Get style recommendations
python D:/kimi/skills/ai-skills/imagen/scripts/imagen.py style --mood professional
```
