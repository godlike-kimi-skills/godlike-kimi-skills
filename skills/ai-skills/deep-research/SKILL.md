# Deep Research Skill

Comprehensive research and analysis tool.

```bash

python D:/kimi/skills/ai-skills/deep-research/scripts/research.py "quantum computing applications"

python D:/kimi/skills/ai-skills/deep-research/scripts/research.py --topic "AI safety" --output report.md
```
